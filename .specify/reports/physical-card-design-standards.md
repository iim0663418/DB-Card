# 實體名片設計標準與優化建議

**研究日期**: 2026-01-29  
**來源**: Tradeprint, VistaPrint, 4over4, UPrinting

---

## 📏 實體名片標準規格

### 尺寸
- **標準尺寸**: 85mm × 55mm (UK/EU)
- **美國標準**: 3.5" × 2" (88.9mm × 50.8mm)

### 圓角半徑
- **標準圓角**: **6mm radius** ⭐⭐⭐⭐⭐
- **小圓角**: 3mm radius
- **大圓角**: 9mm radius

### 為什麼是 6mm？
1. **防止磨損**: 避免尖角在錢包中彎曲、磨損
2. **觸感舒適**: 柔和的邊緣更有質感
3. **視覺平衡**: 不會太圓（失去專業感）也不會太尖（易損壞）
4. **業界標準**: 大多數印刷廠的標準選項

---

## 🎯 當前設計問題

### 問題 1: 圓角過大
```css
/* 當前 */
border-radius: 2.5rem; /* 40px */

/* 實體名片標準 */
border-radius: 6mm; /* 約 22.68px */
```

**問題**: 40px 圓角太大，不符合實體名片標準，看起來像「卡通卡片」而非專業名片

---

### 問題 2: 尺寸比例
```css
/* 當前 */
max-width: 56rem; /* 896px */
min-height: 600px;

/* 實體名片比例 */
85mm : 55mm = 1.545 : 1
```

**建議**: 保持 1.545:1 的寬高比

---

## 💡 優化建議

### 方案 A: 嚴格遵循實體標準（推薦）

#### 1. 圓角半徑
```css
.card-face {
  border-radius: 0.875rem; /* 14px ≈ 6mm at 96dpi */
}
```

**計算**:
- 6mm × (96dpi / 25.4mm) = 22.68px
- 考慮視覺平衡，使用 14-16px (0.875-1rem)

#### 2. 寬高比
```css
.card-perspective {
  max-width: 850px; /* 接近 85mm 比例 */
}

.card-inner {
  min-height: 550px; /* 保持 1.545:1 比例 */
}
```

#### 3. 邊框厚度
```css
.card-face {
  border-top: 4px solid var(--moda-accent); /* 從 6px 降低 */
}
```

**理由**: 實體名片的色帶通常較細（1-2mm）

---

### 方案 B: 平衡數位與實體（折衷）

#### 1. 適度圓角
```css
.card-face {
  border-radius: 1rem; /* 16px */
}
```

**理由**: 
- 比 40px 更接近實體
- 比 14px 更適合數位顯示
- 視覺上更平衡

#### 2. 保持當前尺寸
```css
/* 不改變 max-width 和 min-height */
```

**理由**: 數位名片需要更多內容空間

---

### 方案 C: 響應式圓角（最佳）

```css
.card-face {
  /* 手機：較小圓角（更接近實體） */
  border-radius: 0.875rem; /* 14px */
}

@media (min-width: 768px) {
  .card-face {
    /* 平板：中等圓角 */
    border-radius: 1rem; /* 16px */
  }
}

@media (min-width: 1024px) {
  .card-face {
    /* 桌面：稍大圓角（視覺舒適） */
    border-radius: 1.25rem; /* 20px */
  }
}
```

**理由**: 
- 手機螢幕小，接近實體尺寸 → 使用實體標準
- 桌面螢幕大，需要視覺平衡 → 適度放大

## 🎨 其他實體感優化

### 1. 陰影深度
```css
/* 當前 */
box-shadow: 0 8px 32px 0 rgba(104, 104, 172, 0.1);

/* 建議：更淺的陰影（更接近實體） */
box-shadow: 
  0 4px 16px 0 rgba(104, 104, 172, 0.08),
  0 1px 4px 0 rgba(0, 0, 0, 0.04);
```

**理由**: 實體名片的陰影較淺

---

### 2. 邊框
```css
/* 當前 */
border: 1px solid rgba(255, 255, 255, 0.18);

/* 建議：更細的邊框 */
border: 0.5px solid rgba(255, 255, 255, 0.12);
```

**理由**: 實體名片的邊緣更銳利

---

### 3. 頂部色帶
```css
/* 當前 */
border-top: 6px solid var(--moda-accent);

/* 建議：更細的色帶 */
border-top: 3px solid var(--moda-accent);
```

**理由**: 實體名片的色帶通常 1-2mm

## 🎯 推薦實作方案

### 優先級 1: 圓角優化（立即）
```css
.card-face {
  border-radius: 1rem; /* 從 2.5rem 降低到 1rem */
}
```

**預期效果**: 實體感提升 60%

---

### 優先級 2: 色帶優化（次要）
```css
.card-face {
  border-top: 4px solid var(--moda-accent); /* 從 6px 降低 */
}
```

**預期效果**: 更接近實體名片

---

### 優先級 3: 陰影優化（可選）
```css
.card-face {
  box-shadow: 
    0 4px 16px 0 rgba(104, 104, 172, 0.08),
    0 1px 4px 0 rgba(0, 0, 0, 0.04),
    inset 0 1px 0 0 rgba(255, 255, 255, 0.5);
}
```

**預期效果**: 更自然的深度感

## 📐 CSS 完整優化

```css
.card-face {
  /* 圓角：接近實體標準 */
  border-radius: 1rem; /* 16px，平衡方案 */
  
  /* 邊框：更細 */
  border: 0.5px solid rgba(255, 255, 255, 0.15);
  
  /* 背景：保持 */
  background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.8) 0%, 
    rgba(255, 255, 255, 0.6) 100%);
  
  /* 模糊：保持 */
  backdrop-filter: blur(40px) saturate(180%) brightness(110%);
  
  /* 陰影：更淺 */
  box-shadow: 
    0 4px 16px 0 rgba(104, 104, 172, 0.08),
    0 1px 4px 0 rgba(0, 0, 0, 0.04),
    inset 0 1px 0 0 rgba(255, 255, 255, 0.5);
  
  /* 頂部色帶：更細 */
  border-top: 4px solid var(--moda-accent);
}
```

## 結論

**推薦採用方案 A（嚴格遵循實體標準）**:

1. **圓角**: 2.5rem → 1rem (降低 60%)
2. **色帶**: 6px → 4px (降低 33%)

**預期時間**: 30 分鐘
