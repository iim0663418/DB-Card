## 🎯 WCAG 2.1 AA 標準檢查

### 最小文字大小建議
- **正文**: 最小 16px (1rem)
- **小字**: 最小 12px (0.75rem)
- **極小字**: 最小 10px（僅限非關鍵資訊）

### 當前問題
1. ❌ **表單標籤 10px** - 低於建議最小值
2. ❌ **按鈕文字 10px** - 低於建議最小值
3. ⚠️ **Email 顯示 9px** - 過小，難以閱讀

---

## 📱 響應式考量

### 桌面 (≥768px)
- 當前設定適中，但標籤偏小

### 平板 (640-767px)
- 需要測試觸控友善性

### 手機 (<640px)
- **10px 文字在小螢幕上難以閱讀**
- 建議最小 12px

---

## 🔧 建議調整方案

### 方案 A：最小調整（符合 WCAG AA）
```
表單標籤: text-[10px] → text-xs (12px)
按鈕文字: text-[10px] → text-xs (12px)
Email 顯示: text-[9px] → text-[10px] (10px)
```

### 方案 B：最佳實踐（推薦）
```
表單標籤: text-[10px] → text-sm (14px)
按鈕文字: text-[10px] → text-sm (14px)
Email 顯示: text-[9px] → text-xs (12px)
區塊標題: text-xs → text-sm (14px)
```
