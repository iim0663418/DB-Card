💡 請使用完整的圖片網址：
• Imgur - 上傳後右鍵複製圖片網址
• GitHub - 上傳到 repository 後使用 raw 連結

生成的 NFC 連結

使用說明：
1. 複製上方生成的連結
2. 使用 NFC 寫入工具（如 NFC Tools）將連結寫入 NFC 卡片
3. 使用者觸碰 NFC 卡片即可開啟數位名片

💡 優化建議：
• NFC 卡片容量限制：492 bytes
• 精簡格式已針對 NFC 容量優化
• 如超出限制，請減少問候語或社群備註內容
